üìñ Methodology Explanation

Hybrid N-BEATS + K-Means Process:

  • Step 1 - Load N-BEATS Predictions: Pre-trained N-BEATS model generates predictions for all countries and stores them in cache
  • Step 2 - Select Prediction Year: System retrieves prediction value for the year selected by user (1-5 years ahead)
  • Step 3 - K-Means Clustering: K-Means algorithm performs clustering based on those prediction values
  • Step 4 - Visualization: Clustering results are visualized on an interactive world map with clear country borders

Differences with ARIMA Hybrid:

  • N-BEATS: Deep learning-based neural network for time series forecasting
  • Global Model: One model for all countries (multi-series), different from ARIMA which has one model per country
  • Clustering: Uses simple K-Means for efficiency, not LSTM Autoencoder
  • Performance: Average MAPE ~19% on test data set

Cluster Interpretation:

  • Cluster 0 (Blue - Pattern A): Countries with certain prediction characteristics from deep learning
  • Cluster 1 (Red - Pattern B): Countries with different prediction patterns
  • Cluster 2 (Yellow - Pattern C): Countries with third prediction characteristics

Advantages of This Method:

  • Deep Learning: Uses neural network architecture for more complex forecasting
  • Pre-cached: Predictions are pre-calculated for fast web performance
  • Interactive: Users can select prediction year and see different clustering
  • Visual Clarity: Clear country borders for better geographic identification
