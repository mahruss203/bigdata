üìñ Visualization Explanation

K-Means Clustering is an unsupervised machine learning algorithm that groups data into several clusters based on characteristic similarity.

World Map Interpretation:

  • Country Color: Each country is colored based on its cluster (same cluster = same color)
  • Hover/Click: Point cursor to a country to see name and cluster number
  • Zoom & Pan: Use mouse scroll to zoom in/out, drag to move the map
  • Color Scale: Color scale on the right side shows cluster numbers
  • Geographic Patterns: Notice whether countries in the same cluster are geographically close

How to Use the Map:

  • Move mouse to a country to see detailed information
  • Use scroll to zoom in/out on a specific area
  • Click and drag to move map position
  • Notice color distribution to understand clustering patterns
