üìñ Visualization Explanation

ARIMA (AutoRegressive Integrated Moving Average) is a statistical model used for time series data analysis and forecasting. This model uses Auto ARIMA to automatically find optimal parameters.

ARIMA(p, d, q) Components:

  • p (AR - AutoRegressive): Model that uses the relationship between observations with a number of lagged observations
  • d (I - Integrated): Uses differencing to make data stationary
  • q (MA - Moving Average): Model that uses the dependency between observations and residual errors

Application Features:

  • Select Country: Choose the country you want to analyze from the dropdown (194 countries available)
  • Custom Prediction Years: Determine the number of future years to predict (1-20 years)
  • Auto Parameter Selection: The model automatically determines the best ARIMA parameters for each country
  • Accuracy Evaluation (MAPE): Displays Mean Absolute Percentage Error to measure model accuracy

Visualization Interpretation:

  • Solid blue line: Displays actual historical stroke deaths data per year
  • Dashed orange line: Displays ARIMA model prediction results for future years
  • Vertical red line: Marks the boundary between historical data and predictions
  • Markers (o and ‚ñ°): Show data points for easier reading
  • Trend: General pattern of data movement over time - up, down, or stable

ARIMA Model Uses:

  • Forecast future stroke deaths based on historical patterns per country
  • Identify trends and patterns in time series data
  • Assist in health planning and medical resource allocation
  • Evaluate and monitor health performance over time
  • Cross-country comparisons for stroke prevention strategies
